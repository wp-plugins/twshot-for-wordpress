Quickly add your post links to twshot and update your twitter statues with this twshot link

== Description ==

Twshot is a web service that lets you save & post your long links to twitter by cutting them. 

Twshot for WordPress is a plugin that adds your posts's link to twshot for you when you publish them. It saves your time when you publish a post or page.

If you use plugin you don't have to update your twitter status manually, plugin will automaticaly update it for you.

== Installation ==

1. Upload `twshot-for-wordpress` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go your Options/Twshot section in your admin panel. Type your twitter and username correctly then save it.
4. In post write page you will see a metabox that has a field labeled Twshot title, type your message that you want it to update your twitter status.
5. Whenn you publish your post or page your twitter status will be updated with twshot title + twshot url
6. That's it :)
